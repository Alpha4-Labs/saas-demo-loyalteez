# Contributing to SaaS Demo

Thank you for your interest in contributing to the SaaS Demo! This document provides guidelines and instructions for contributing.

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Getting Started

1.  **Fork the repository**.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/your-username/saas-demo-loyalteez.git
    ```
3.  **Install dependencies**:
    ```bash
    npm install
    ```
4.  **Create a branch** for your changes:
    ```bash
    git checkout -b feature/your-feature-name
    ```

## Development Workflow

1.  Make your changes.
2.  Test your changes locally.
3.  Commit your changes with clear messages.
4.  Push to your fork.
5.  Open a **Pull Request** (PR) to the `main` branch.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub.
